# Flappy bird Clone on Clojure CLR (.NET) using monogame

## Build

```
dotnet tool restore
dotnet build
```

## Running on Dotnet

Compile and run:

```bash
dotnet run
```

## Running on ClojureCLR

Run:

```bash
dotnet tool restore
dotnet Clojure.Main -m cljgame.game
```

More info here:
https://github.com/clojure/clojure-clr/wiki/Getting-started
